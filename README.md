# Electricity Sector CO2 Emissions Analysis

This project analyzes electricity-sector CO2 emissions in Georgia and compares them with Tennessee, Texas, California, and the United States using EIA State Energy Data System (SEDS) data.

## Tools
- R
- ggplot2
- tmap
- dplyr

## Key Visualizations
- State-level CO2 emissions map
- Fuel mix comparison
- CO2 emissions comparison
